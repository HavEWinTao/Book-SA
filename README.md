# book(gauss)-java

https://github.com/HavEWinTao/Book-Gauss-Java.git

## How to run

1. 安装JDK1.8
2. 安装Maven并设置settings.xml
3. 修改设置

   修改idea中的`Maven home`,`User settings xml`和`repository`的路径为自己安装的maven的路径
4. 运行Application
